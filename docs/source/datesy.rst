The datesy package
==================

The `datesy` package is divided in 4 main components:

1. `file I/O (subpackage) <https://datesy.readthedocs.io/en/latest/datesy.file_IO.html>`_
2. `data converting <https://datesy.readthedocs.io/en/latest/datesy.html#module-datesy.convert>`_
3. `data inspecting <https://datesy.readthedocs.io/en/latest/datesy.html#module-datesy.inspect>`_
4. `matching data sets <https://datesy.readthedocs.io/en/latest/datesy.html#module-datesy.matching>`_
